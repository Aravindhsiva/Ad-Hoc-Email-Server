AHEM: Ad-Hoc Temporary Email Address

AHEM is an ad-hoc Email server.
Ad-hoc means that a mail that a mailbox is created for you as soon as you need it. Others call it disposable mail or a temporary email address.

What is an AHEM disposable temporary Email address?

A temporary disposable Email service allows you to receive mail at a temporary mail address created ad-hoc, when you need it.
You can use a disposable Email service to avoid exposing your real email when using services that require Email registration, such as wifi providers, forums or other services.
In addition, using disposable mailboxes is a great way to test software that relies on email.
AHEM even provides a very simple and powerful RESTful API to consume Email.

AHEM Email Server Features

  • Prevent spam - use an AHEM mailbox when signing up to wifi networks, forums and sites and prevent unwanted spam.
  • Software testing - need for an ad hoc temporary disposable mailbox?
  • Install an on prem version of this services easily!
  • Emails are automatically deleted after a few hours.
  • Auto generated or selected email address.
